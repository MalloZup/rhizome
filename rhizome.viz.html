rhizome.viz documentation

create-frame

fn

[{:keys [name close-promise dispose-on-close?]}]

Creates a frame for viewing graphviz images. Only useful if you don't want to use the default frame.

dot->image

fn

[s]

Takes a string containing a GraphViz dot file, and renders it to an image. This requires that GraphViz is installed on the local machine.

dot->svg

fn

[s]

Takes a string containing a GraphViz dot file, and returns a string containing SVG. This requires that GraphViz is installed on the local machine.

save-graph

fn

[nodes adjacent & {:keys [filename], :as options}]

Takes a graph descriptor in the style of graph->dot, and saves the image to disk.

save-image

fn

[image filename]
[image filetype filename]

Saves the given image buffer to the given filename. The default file type for the image is png, but an optional type may be supplied as a third argument.
